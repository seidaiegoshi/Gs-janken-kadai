# 使用技術

### JavaScript

- ### `canvas`

  html の`<canvas>`タグに JavaScript で描画を行う

- ### `setInterval()`

  繰り返し処理を行う

- ### `Math`
  乱数生成(`random`)、距離角度の計算(`sin` `cos`)などで使用

---

# 超ざっくりゲームの仕組み

- 10msec 毎に描画、位置計算や当たり判定などの処理を行う
- 各ステージには wave があり、各 wave は敵の手の配列を持つ
- 1 つの手は`位置`,`スピード`,`角度`,`手の種類`の情報を持つ
